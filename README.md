# CodeCounter-rs

I was kinda bored and thought it might be funny lol, so I built a tiny Rust TUI that counts the lines of code in whatever directory you run it in, then yells the total back at you in big ASCII art.

uses [Ratatui :D](https://ratatui.rs/) <img width="32" height="32" alt="image" src="https://github.com/user-attachments/assets/4f44983a-d88d-4a91-8fa1-4013b82fc567" />

## Usage
```sh
codecounter
```
